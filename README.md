# Loan Default Analysis Dashboard (Power BI)

## Objective

Developed an interactive dashboard to analyze loan approval and default trends, helping identify high-risk customers.

## 🛠️ Tools Used

* Power BI
* Data Visualization
* DAX (Basic KPIs)

## 📊 Dataset

* Loan dataset with 2000+ records including credit score, loan amount, approval status, and default status

##  Key Analysis

* Calculated KPIs such as approval rate, default rate, and total loan amount
* Identified high-risk customer segments based on credit score
* Analyzed loan distribution and approval patterns
* Performed trend analysis on defaults

## Dashboard Features

* Interactive visuals with filters and slicers
* KPI cards for quick insights
* Dynamic filtering for better analysis

## Files Included

* `Loan_Dashboard.pbix` – Power BI file
* `dashboard.png` – Dashboard screenshot
